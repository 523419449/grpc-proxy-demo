## GRPC简介

GRPC是一个现代的开源高性能RPC框架，可以在任何环境中运行。它可以高效地在数据中心内部和跨数据中心连接服务，并支持可插拔的负载平衡、跟踪、运行状况检查和身份验证。它也适用于分布式计算的最后一英里，将设备、移动应用程序和浏览器连接到后端服务。
